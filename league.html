League of Legends | Zodiac Calendar

Front-End Development Case Study

The goal of this project was to create a website which showcases 12 different Zodiac signs using different League of Legends characters.

Problem Statment

The project was created to provide a fun, interactive way for users to discover which League of Legends champion aligns with their zodiac sign based on their birthday. There was no existing tool combining zodiac signs and LoL champions so this project aimed to fill that gap.

My Role

This was a solo project where I handled everything from initial design to final deployment. I created wireframes and a prototype in Figma, then built the site using HTML, CSS, and JavaScript. I was responsible for both the design and development.

Technology Used

The site was built using HTML for structure, CSS for styling and layout, and JavaScript to handle user input and zodiac logic. Figma was used for designing the interface, and all code was managed and versioned through GitHub. No additional frameworks or libraries were used.

Results & Outcome

The final product successfully delivered an interactive tool that matched users to champions based on their birthday. It met its goals of combining entertainment with clean UI/UX. A key takeaway was how effective simple logic and intuitive design can be in creating engaging user experiences.

Typography

For this project, I used Beaufort, the same font family used in League of Legends. It helps keep the branding consistent and adds to the overall feel of the site. The bold and elegant style fits perfectly with the game's fantasy vibe and strategic tone, making the experience feel more connected and immersive.

Color

The colors used in this project are pulled directly from League of Legends’ official palette to maintain brand authenticity and enhance immersion. Gold (#C89B3C) highlights key info, while muted tan (#A09B8C) keeps text readable with a fantasy feel. Dark navy (#010A13) sets a deep background tone.
