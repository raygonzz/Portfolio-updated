Bergo Fencing Inc. Website Redesign

Front-End Development Case Study

The goal of this website redesign is to revamp it with a new feeling. The old design is outdated and frustrating to navigate.

Problem Statment

The website redesign for Bergo Fencing Inc. was necessary largely due to the outdated look, excessive white space, and poor mobile usability of the original site. The goal was to create a modern, responsive design that better showcased the company's services, improved user experience, and encouraged customer inquiries. The project focused on redesigning key pages, optimizing layout and images, and ensuring mobile compatibility.

My Role

This was a solo project where I was responsible for the entire redesign process from start to finish. I created wireframes, designed a high-fidelity prototype in Figma, and then programmed the fully responsive website based on that design. All design and development decisions were made independently.

Technology Used

The project utilized HTML and CSS for building and styling the website, providing full control over layout and responsiveness. Figma was used during the design phase to create wireframes and a high-fidelity prototype, allowing for clear visualization before development began. GitHub served as the version control and code repository platform, ensuring organized progress tracking and easy access to previous iterations.

Results & Outcome

In the end, the project was successfully completed, and the goals were achieved. The website now features a modern, responsive design that effectively showcases the contractor's services and provides a much-improved user experience. One key lesson learned was the importance of planning for mobile-first design to ensure accessibility across all devices. Additionally, I gained valuable experience managing the entire project solo, from design through to development, which improved my skills in both areas.

Typography

This project uses a two-font system to create clear visual hierarchy and enhance readability. Noto Sans was selected for headers and emphasis text due to its clean, modern appearance and strong legibility, helping key elements stand out. For body text and longer paragraphs, Lexend Light was chosen for its soft, open letterforms, promoting a smoother reading experience and maintaining a clean, minimalist tone throughout the design.

Color

For this project, I chose a bold and modern color palette to create visual impact and maintain strong contrast for accessibility. The primary red (#FF464A) brings energy and draws attention to key interactive elements, while the deep gray (#3D3D3D) provides a sleek, neutral background that enhances readability and reduces eye strain. The crisp white (#FFFFFF) ensures clarity and clean separation of content, reinforcing a minimalist and user-friendly aesthetic.
